Informations about this site :

The goal of this website is to allow you to find co-occurences between two biological entities.
Biological entities can be species, cellular types, mutations, histones modifications, genes, enzymes...
In order to make your query you have to click on the corresponding button in the navbar.
Your query will be directed against the site's database, which structure is explained down below.

Informations about the database :

This site is based on the GeneView database (2018), most tables contain these 3 collumns to conduct your research:

pmid :
the PubMed identifier of the article the biological entity appears.
entity :
the name of the biological entity, be careful about these names as biological entities may have many names, such as "Gamma-glutamyl transpeptidase" can be refered as "gamma-GT".
normalized :
this field can come in handy to manage entities in the above cases. This field contains the normalized denomination for biological entities,
so both the "gamma-GT" and "Gamma-glutamyl transpeptidase" share the normalized name of "2678".

Pictures of database tables are available in the "Make a query" web page for a visual understanding of the matter.

Abbreviations : This table contains both the full name of the biological entity and its abbreviations.
Cell : Use this table if you wish to look for a specific cellular type.
Chemicals : This table contains chemical compounds.
Dates : This table contains the dates of publication of each article in the database.
Disease : This table contains the name of diseases as well as symptoms (e.g. nausea).
Enzyme : Use this table if you wish to look for a specific enzyme.
Genes : Use this table if you wish to look for a specific gene.
Histones : Use this table if you wish to find histone modifications.
Mutations : This table contains any type of mutation mentionned in the articles withing GeneView.
Species : Use this table if you are looking for a specific specie.
Tissue : Use this table if you are looking for a specific tissue.
